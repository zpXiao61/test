Sophon Inference 安装¶

针对不同的应用场景，BITMAIN基于BM168x芯片提供了PCIE和SOC两种产品形态，对应于两种不同的运行模式：

  • PCIE模式：SDK运行于X86平台，BM168x作为PCIE接口的深度学习计算加速卡存在；
  • SOC模式：SDK独立运行于BM168x平台，支持通过千兆以太网与其他设备连接。

BMNNSDK支持以上两种运行模式，在应用接口上完全兼容，请根据产品形态和需求下载对应的BMNNSDK进行安装。 需要注意的是，在 SOC 模式下，BMNNSDK 中不包含 BMCompiler 模块， 您可在 PCIE 模式下使用 BMCompiler 完成模型的离线编译(不需要安装 SC3 加速卡)， 编译生成的 bmodel 可以在 SOC 模式下加载并运行。

目前 SophonInference 支持 PCIE 模式与SOC 模式。 对于 SOC 模式，Sophon Inference 已经预装在 Sophon SE3 产品上， 包含了 SAIL(cpp) 的头文件与链接库文件以及 SAIL、AutoRunner、Algokit(python3)等python模块。 对于 PCIE 模式，我们也提供了 SAIL (cpp) 的头文件与链接库文件以及 SAIL、AutoSplit、AutoRunner、Algokit(python3) 等python模块。

PCIE 模式¶

a).依赖

硬件环境：安装了 Sophon SC3 加速卡的 X86 主机

操作系统：Ubuntu/CentOS/Debian

软件环境：安装了 PCIE 模式的 BMNNSDK

b).安装

# 安装 Sophon Inference 的 python 包
cd ${sophon_inference}/python3/x86
pip3 install sophon_x86-1.1.3-py3-none-any.whl --user
